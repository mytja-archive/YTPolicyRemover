YTPolicyRemover



YTPolicyRemover is a little project by @mytja, that helps you remove all annoying boxes that pop up from YouTube GUI, like ToS box and login box, which means, you'll get a smooth experience without any popups!
It is completly free, can be used with Firefox, Chrome, Opera, Edge, Brave browser...
So why don't you try it 😀
